# weather-app

## Features

- Search for location
- Autocomplete search
- Get weather data from API and display current conditions and 10-day forecast
- Display icons based on weather conditions
- Toggle units (&deg;C/kph or &deg;F/mph)

## Built with

- [TypeScript](https://www.typescriptlang.org/)
- JavaScript
- HTML
- CSS
